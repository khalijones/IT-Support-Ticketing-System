# IT-Support-Ticketing-System
Full-stack ticketing system with role-based access, SLA tracking, and knowledge base — reflecting real support workflows from Swift Response.

Getting Started
1. Install dependencies:
    cd backend && npm install
    cd ../frontend && npm install

2. Start MongoDB
     # Using Docker
     docker-compose up -d mongodb

3. Run the application:
     Terminal 1 - Backend
     cd backend && npm run dev

     Terminal 2 - Frontend
     cd frontend && npm start
